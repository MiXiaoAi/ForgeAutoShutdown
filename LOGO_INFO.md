# Logo 说明

## 文件说明

- `logo.svg` - 完整版 logo（128x128），用于 README 和文档
- `icon.svg` - 简化版图标（64x64），用于社交媒体等小尺寸场景

## 设计元素

### 主要元素
- **电源按钮**：代表服务器关闭功能
- **时钟**：代表定时功能
- **服务器图标**：代表 Minecraft 服务器
- **像素装饰**：体现 Minecraft 风格

### 颜色方案
- 主色：`#2D5A87` (深蓝)
- 强调色：`#4A90E2` (亮蓝)
- 辅助色：`#87CEEB` (天蓝)
- 深色：`#1E3A5F` (深蓝灰)

## 使用场景

- GitHub README
- 发布页面
- 文档封面
- 社交媒体分享

## 制作工具

Logo 使用 SVG 格式制作，可以无损缩放到任意尺寸。如需 PNG 格式，可使用以下工具转换：
- Inkscape (免费)
- Adobe Illustrator
- 在线 SVG 转 PNG 工具
